# Color Clock

A bedside clock app for kids where screen colors signal daytime vs nighttime.

Parents set a day/night schedule and pick colors for each mode. At night the
screen goes dim with calming colors (e.g. deep blue + dim red digits). When
morning comes, the clock switches to bright cheerful colors (e.g. light yellow +
green digits), telling the child it's okay to get up.

## Features

- Full-screen digital clock, landscape orientation
- Day and night modes with independent colors and brightness
- Screen stays on (wake lock), no screen timeout
- Per-window brightness control (very dim at night, bright during day)
- 5 editable color presets per setting, with full HSV color picker
- Configurable 12h/24h time format with AM/PM
- Immersive mode (no status bar or nav bar)
- Settings auto-save, persist across restarts
- Nunito Bold font for friendly, readable digits

## Getting started

1. Open the project in Android Studio
2. Sync Gradle
3. Run on an emulator or connect an Android device via USB
4. For standalone install: Build > Build APK(s), transfer to device

## Requirements

- Android 7.0+ (API 24)
- Android Studio with Kotlin and Compose support
